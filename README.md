# RFP Viewer

A Next.js workbook for managing and responding to RFP (Request for Proposal) questions. Currently configured for the BSB Credit Card RFP with Brim Financial.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features
- Grid view of RFP questions and responses
- Context panel for question details
- Rules panel for response guidelines
- Export to JSON and CSV
